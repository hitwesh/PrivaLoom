# Frontend Upload Test Files

This folder contains synthetic files for testing dataset uploads from the frontend.

Suggested quick test flow:
1. Open frontend workspace.
2. In Dataset Intake, click Select dataset files.
3. Pick 2-4 files from this folder.
4. Open Selection tab and keep files selected.
5. Click Send updates.

Notes:
- All files are fake sample content.
- Formats included: TXT, CSV, JSONL.
- You can upload any combination.
